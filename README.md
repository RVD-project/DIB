## 🐶 About this Service 

DIB은 원하는 지역사람과 자유롭게 공동구매를 진행할 수 있는 웹 서비스입니다. 사용자들은 파티를 생성하거나 참여하여 대용량 물품을 구매하고 만남을 약속하여 거래를 진행할 수 있습니다.


## ✨ Features


MVP에서 다음과 같은 기능을 완성했습니다.

- ✅ 카카오 로그인 기반 회원가입
- ✅ 파티 생성 및 참여
- ✅ 별점, 후기 시스템
- ✅ 파티원들과의 실시간 채팅 기능

개발기간 : 2023/08/09 ~ 2023/09/13
